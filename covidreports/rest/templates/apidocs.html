API Documentation

This project now has a RESTful component to it. This API will essential act as a wrapper around the third party APIs that this website gets its data from, as well as allow users to access the same data rendered in the tables via JSON.

All API endpoints use the GET method, free to use, and do not require a key.


World Data Endpoints

  • /api/world/summary/: Summary of current data for all countries.
  • /api/world/country_name/: COVID case history for a specific country.
  • /api/world/percentages/: What percentages of global cases, deaths, etc different countries constitute.

U.S. Data Endpoints

  • /api/us/summary/: Summary of all current data for each U.S. state/territory.
  • /api/us/state/state_name/: A history of COVID data for a specific state.
